Meus Projetos

Nike Air Zoom – Landing Page

Landing page desenvolvida para destacar o modelo Nike Air Zoom, com foco em design clean, boa experiência do usuário e apresentação clara do produto. O layout valoriza a imagem do tênis, apresenta variações de cor e inclui botão de compra com destaque.

Projeto iV Store – Interface moderna e responsiva com carrossel interativo

O projeto foi uma oportunidade de colocar em prática os conhecimentos adquiridos em HTML, CSS e JavaScript puro, aplicando conceitos fundamentais de estruturação, estilização e lógica. A interface apresenta um carrossel interativo de produtos, com transições suaves, responsividade e um design focado na experiência do usuário. Cada detalhe foi pensado para tornar a navegação leve, visualmente agradável e funcional, reforçando minha evolução como desenvolvedor front-end.

CODE VERSE – Uma Jornada Digital Interativa

Com HTML, CSS e JavaScript, dei vida a uma jornada visual que se movimenta com o scroll, trazendo animações 3D e interações que envolvem o usuário logo nos primeiros segundos. Cada detalhe foi pensado para transmitir a essência da comunidade CODE VERSE: um espaço onde pessoas compartilham conhecimento, constroem juntos e acreditam no poder do código para transformar o mundo.
